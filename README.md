# Deadbounce

One-thumb neo-western neon arena shooter. Bullets do zero damage on a direct
hit — they only become lethal after ricocheting off walls, and every bounce
adds damage and speed.

Flutter + Flame + Cubit (flutter_bloc), clean architecture.

See [SETUP.md](SETUP.md) for how to run, configuration, and what is
intentionally stubbed in this foundation pass.
